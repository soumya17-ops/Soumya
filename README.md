# 🌐 Personal Portfolio Website

## 👩‍💻 About the Project

This is my personal portfolio website built using **HTML, CSS, and JavaScript**.
It showcases my skills, education, and projects in a clean and responsive design.

The website is designed to present my profile professionally and help recruiters understand my technical background and work.

---

## 🚀 Features

* Responsive and modern UI design
* Smooth animations using AOS (Animate On Scroll)
* Navigation bar with smooth scrolling
* Sections included:

  * Home
  * About
  * Education
  * Projects
  * Skills
  * Contact
* Social media links (GitHub, Instagram, LinkedIn)
* Mobile-friendly hamburger menu

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* AOS Library (for animations)
* Boxicons (for icons)
